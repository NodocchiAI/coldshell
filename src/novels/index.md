# 深海エリア データログ

## 読み方ガイド

1. 気になるタイトルをクリック。  
2. 開いた記事で `Deep-LINK` コードをコピーすると、お友だちにURLをメールできます。  
3. 読了後は「感想ポスト」リンクからBBSへどうぞ。
