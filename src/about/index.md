# Navigation

黒潮湾市立水族館をご利用いただく皆さまへ。ここでは館内フロア構成と巡回ルート、サービス窓口の情報を旧式端末用に集約しています。スクロールしながら必要なセクションを確認してください。

## floremap概要

1. **ネオンロビー (1F)**
   - チケットカウンター、LED掲示板、記念撮影スタンドを設置。
   - ここでBBSパスワード入りパンフレットを受け取ってください。
2. **水槽エリア (2F)**
   - サイバー珊瑚トライデッキ、潮流バーチャルリングなどの一般展示。
   - ガイド端末で各水槽のリアルタイムデータを閲覧できます。
3. **深海エリア (B1)**
   - ブラックアウトホール、ソナーシアター等を含む薄明ゾーン。
   - 入場時は暗闇に慣れるまでスタッフが同伴します。

## 推奨ルート

| ステップ | 所要時間 | 内容 |
| --- | --- | --- |
| Step.1 | 15分 | ネオンロビーでイベント情報を確認し、館内回線(56kbps)へ接続。 |
| Step.2 | 30分 | 水槽エリアでインタラクティブ展示を体験。潮流バーチャルリングの操作訓練を推奨。 |
| Step.3 | 20分 | 深海エリアへ降り、プラズマクラゲ回廊を鑑賞。フォトスポットでGIF撮影。 |
| Step.4 | 10分 | ミュージアムショップでASCIIポストカードを購入し、出口スタンプを押印。 |

## サービス案内

- **掲示板カウンター**: 受付横に常設。来館者のメッセージをスタッフが手入力でBBSへ転載します。
- **音声ガイド**: カセットウォークマン型デバイスを貸し出し（日本語/英語/分類不能言語）。
- **休憩スペース**: 56kbps Wi-Fiとレトロ自販機を完備。飲み物の自動釣銭は少し遅いので気長にお待ちください。
- **非常時**: 緊急シャッター作動時は係員が誘導します。警告灯が赤に変わったら指示に従ってください。
